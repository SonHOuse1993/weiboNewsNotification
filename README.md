# weiboNewsNotification

微博信息提示助手

## 主要内容

我想把关注的人的微博 变成一条条信息 看完一条点一下 变灰一条（内容包括评论）

如果可以朗读就更好了

毕竟我只是想换一种形式显示浏览器看到的内容
